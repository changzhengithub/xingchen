# 快速查看vue build 后的体积大小

使用插件 vue-cli-plugin-webpack-bundle-analyzer

安装： vue i vue-cli-plugin-webpack-bundle-analyzer -D

安装好之后不需要任何配置直接 运行命令

 	npm run build -- report
